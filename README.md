# User Management System

## Features

* Multi-Step User Registration
* JWT Authentication
* Login System
* Admin Dashboard
* User Search
* Delete User Functionality
* Responsive UI using Bootstrap

## Tech Stack

### Frontend

* React
* Axios
* Bootstrap

### Backend

* Node.js
* Express.js
* JSON Web Token (JWT)

## How to Run

### Backend

```bash
cd server
npm install
node server.js
```

### Frontend

```bash
cd client
npm install
npm start
```
